## Анализ выполненной работы по кейс-задаче № 3

## 1. Выбранная кейс-задача

**Кейс-задача № 3:**  
Написать программу с использованием языка программирования Python, которая будет генерировать последовательность случайных чисел до тех пор, пока пользователь не введёт ноль. Вывести на экран все числа, кроме последнего.

---

## 2. Исходный код программы

```python
import random

nums = []

while True:
    choose = int(input("Введите 0 для остановки или любое другое число для генерации: "))

    if choose == 0:
        break

    number = random.randint(1, 100)
    nums.append(number)

print("Сгенерированные числа, кроме последнего:")

if len(nums) > 1:
    print(nums[:-1])
else:
    print("Нет чисел для вывода")
```

---

## 3. Описание работы программы

Программа предназначена для генерации последовательности случайных чисел. Пользователь управляет процессом генерации через ввод с клавиатуры. Если пользователь вводит любое число, кроме нуля, программа генерирует случайное число в диапазоне от 1 до 100 и добавляет его в список. Если пользователь вводит `0`, цикл завершается.

После завершения генерации программа выводит все сгенерированные числа, кроме последнего. Если было сгенерировано меньше двух чисел, программа сообщает, что чисел для вывода нет.

## 4. Анализ выполненной работы

### 4.1. Функциональность

Программа выполняет основную поставленную задачу: генерирует случайные числа до тех пор, пока пользователь не введёт ноль, а затем выводит все числа, кроме последнего.

**Положительные стороны:**

- используется цикл `while True`, который позволяет продолжать генерацию до команды остановки;
- условие `if choose == 0` корректно завершает работу цикла;
- случайные числа сохраняются в список;
- при выводе применяется срез `nums[:-1]`, который исключает последний элемент.

**Выявленная проблема:**

Если пользователь введёт не число, например букву или пустую строку, программа завершится с ошибкой `ValueError`. Это снижает корректность обработки входных данных.

**Способ решения:**

Добавить обработку исключений с помощью конструкции `try-except`, чтобы программа не завершалась аварийно при некорректном вводе.

### 4.2. Производительность

Программа работает быстро, так как выполняет простые операции: ввод данных, генерацию случайного числа, добавление элемента в список и вывод результата.

**Положительные стороны:**

- операции добавления в список выполняются эффективно;
- программа не использует сложные вычисления;

**Выявленная проблема:**

Если пользователь сгенерирует очень большое количество чисел, список `nums` может занять значительный объём оперативной памяти.

**Способ решения:**

Для данной учебной задачи использование списка является допустимым. Однако при необходимости обработки большого количества данных можно использовать потоковую обработку или сохранять данные во внешний файл.

## 5. Улучшенный вариант программы

```python
import random

nums = []

while True:
    try:
        user_input = int(input("Введите 0 для остановки или любое другое число для генерации: "))
    except ValueError:
        print("Ошибка: введите целое число.")
        continue

    if user_input == 0:
        break

    number = random.randint(1, 100)
    nums.append(number)

print("Сгенерированные числа, кроме последнего:")

if len(nums) > 1:
    print(nums[:-1])
else:
    print("Нет чисел для вывода")
```
